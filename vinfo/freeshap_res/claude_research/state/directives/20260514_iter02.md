## 사용자 직관 (formalize 할 대상)

(i) **top-r 의 정체**: top-r by λ_i 는 label 정보 무시. 순수 *FC selection* — 즉
  best rank-r approximation of K in Frobenius norm.

(ii) **A1 의 정체 (λ_i >> ρ regime)**: A1 의 score s_i = (λ_i/(λ_i+ρ))²·(u_i⊤Y)²
  에서 filter (λ/(λ+ρ))² ≈ 1 ∀i. → A1 의 selection 은 사실상 *top-r by (u_i⊤Y)²*
  = *max LC selection*. eigenvalue 자체는 "this mode is learnable" gate 역할만.

(iii) **두 방식의 직교성**: FC axis (kernel norm) 와 LC axis (label projection) 는
  서로 독립적 정보. top-r 은 LC 무시, A1 은 FC 무시. *axis 자체의 직교성* 은
  보장되지만, *selection 결과의 직교성* 은 dataset 의 spectrum-label joint
  distribution 에 의존.

(iv) **각 방식이 놓치는 정보의 quantification**:
  - **top-r 이 놓친 정보**: 1 − LC(I_LR) = Σ_{i ∉ I_LR} (u_i⊤Y)² / Σ_j (u_j⊤Y)²
  - **A1 이 놓친 정보**: 1 − FC(I_A1) = Σ_{i ∉ I_A1} λ_i / Σ_j λ_j

## 핵심 분석 질문

**Q1 (formalization)**: 위 사용자 직관 (i)-(iv) 를 식 단위로 정확히 표현. 특히
  - top-r 의 maximization objective vs A1 의 maximization objective 의 식.
  - 두 selection 의 직교성 정도를 *측정 가능한 한 양* 으로 (e.g., overlap, Spearman correlation).
  - "놓친 정보" 의 정확한 식 + 해석 (LRFShap 의 eq. (9) bound 와의 관계).

**Q2 (predictive power)**: precomputed.json 의 quantities 중 어느 것이 dataset 별
  empirical_wins (4/42 ~ 42/42) 와 가장 잘 상관되는가? 가설:
  - 후보 A: 1 − LC(I_LR) (= top-r 이 놓친 label energy). 클수록 A1 가치 큼?
  - 후보 B: 1 − FC(I_A1) (= A1 이 놓친 kernel norm). 클수록 LRFShap 가치 큼?
  - 후보 C: Spearman(λ, c²) — 두 axis 의 alignment.
  - 후보 D: ratio eq9_gap_A1 / eq9_gap_LR.
  - 후보 E: overlap_LR_A1 (1 에 가까울수록 두 방법이 동일 mode 선택, 그러면 차이도 작아짐).
  - 또는 이들의 조합.

**Q3 (interpretation)**: 어느 후보가 가장 좋은지 정량적으로 결정한 뒤, *왜* 그런지
  mechanism 설명. iter 01 의 archetype 분류 (multi-class 우세 등) 와 어떻게 연결되나.

**Q4 (caveats and counterexamples)**: 어떤 dataset 이 가설을 벗어나는가? 예:
  RTE 와 MRPC 모두 high miss_LC_by_LR (~0.79, ~0.72) 인데 win 수가 4/42 vs
  35/42 로 크게 다름. 이건 무엇으로 설명되나? 가설은 "RTE 의 missing LC 가
  *noise* (population alignment 없음) 인 반면 MRPC 의 missing LC 는 *real signal*
  (minority detection)". 이를 *측정 가능한 양* 으로 표현하기는 어려운가?

### critic (round 1, round 2)

**역할**: planner 의 framework + empirical analysis 검증.

**의무 (round 1)**:
1. **Formalization 검증**: 식이 정확한가? 특히 (a) A1 의 score 가 *정말* LC 와
   같은지의 limit 분석 (λ>>ρ 의 정확한 조건), (b) Q1 의 직교성 표현이 dataset 의
   spectrum-label joint distribution 에 어떻게 의존하는지.
2. **Quantitative 검증**: planner 가 선정한 best predictor 척도의 robustness —
   outlier dataset (RTE, MRPC) 가 척도를 부적절하게 보이게 만들지 않나?
3. **숨겨진 가정**: A1 의 evaluation 이 *inv prediction* 으로 측정됐다는 점이
   분석에 어떻게 영향을 주나? (Shapley-side 와 prediction-side 의 분리)
4. **반례 / 검증 실험 제안**: synthetic data 또는 추가 dataset 으로 본 framework
   를 falsify 할 수 있는 실험 1-2 개 제안.
