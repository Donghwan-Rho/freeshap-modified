## 배경 — 핵심 발견 두 가지

**(A) Valbal (train imbalanced + val balanced)**: a1 이 FreeShap (inv) / LR
(top-r eigen, = lrfshap top-r) 의 collapse 를 균일하게 회복한다. small budget
(sel ≤ 5%) 에서 일반적으로 a1 ≥ random.

**(B) Trainbal (train balanced + val imbalanced, 새 setting)**: balanced
accuracy metric 으로 보면 *거의 모든 case* 에서 a1 이 FreeShap/LR 을 압도한다
(이들은 거의 collapse). 그러나 imbalanced cell (70/30 또는 cls60/55, 90/10 또
는 cls90/85) 중 *절반 정도 case* 에서 a1 이 random 한테 패배한다.

같은 setting 인데 imbalance 가 train 측 ↔ val 측으로 옮겨갔을 뿐이다. 왜 a1 의
*상대적 효과* 가 갈라지는가? 그리고 *왜 FreeShap/LR 은 collapse* 하는가?

## Naming (이 iter 부터 일관 적용)

- **FreeShap** = inv approximation (Wang, Lin, Qiao, Foo, Low 2024). full
  kernel rank.
- **LR r%** = lrfshap top-r eigen approximation. r ∈ {1, 5, 10, 15, 20, 25, 30}%.
- **A1 r%** = label-aware top-r eigen approximation. sᵢ = (λᵢ/(λᵢ+ρ))² · ‖uᵢ⊤Ỹ‖².

"inv" 와 "lrfshap" 의 hidden continuity (LR r=100% ≈ FreeShap) 는 §technical
정리에 한 줄로 명시. paper 에서는 별도 method 로 다룸.

## 분석 lens (우선순위)

순서대로 다루되 lens 1-2 는 깊게, 3-4 는 brief.

1. **Label shift / prior shift literature**. Saerens 2002, Lipton 2018 (BBSE),
   Garg 2020. P_train(y) vs P_val(y) 의 mismatch 가 ridge regression 의
   classifier risk 에 미치는 영향. balanced accuracy 와 직결. trainbal 의
   random-loss 도 이 lens 로 설명 가능한지 검증.

2. **Spectral / kernel target alignment**. Bach-Jordan 2002 (KTA), Cristianini
   2001. K vs Ỹ Ỹ⊤ 의 alignment 가 small budget 에서 어떻게 변하는지. LR 의
   top-r eigen 이 *train spectrum* 의 top 이지만 *val Ỹ* alignment 가 낮을 때
   collapse 한다는 가설. A1 의 score 가 이 alignment 를 직접 보정.

3. **Effective dimension d_eff(ρ) / source condition** (brief). Caponnetto-De
   Vito 2007. valbal vs trainbal 에서 d_eff 변화. lens 2 의 sub-lens.

4. **Shapley value variance** (brief). TMC iteration 의 variance. A1 vs LR 의
   variance 비교. lens 1-3 이 insufficient 일 때만 정밀 분석.
